Custom Post Type "Agenda" dengan meta field start_date, end_date, time, location, dan register_url agar dapat dipakai oleh frontend headless.

== Installation ==
1. Upload folder `blc-agenda-cpt` ke direktori `wp-content/plugins/` atau instal melalui ZIP.
2. Aktifkan plugin melalui menu **Plugins** di WordPress.
